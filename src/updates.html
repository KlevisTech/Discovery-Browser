Discovery Browser
Version 3.0.0
Version 3.0.0 is a major Discovery Browser update focused on richer media playback, clearer controls, smoother site cards, safer challenge handling, built-in privacy, and easier guidance for every feature.
Visualizer & Viewing Modes
F
Side Flames New
The visualizer header can add animated flames along the sides of supported Wave and Curve site cards, giving active media a more immersive visual frame. Use the Flames control to turn the effect on or off.
C
Cinema Mode New
Cinema Mode expands the active site video into a distraction-free viewing layout while keeping it inside Discovery Browser. Unsupported or black-screen sites are detected and a compatibility message explains why the mode could not be used.
Full
Full Mode New
Full Mode asks the active video to use its native full-video presentation. When native fullscreen is unavailable, Discovery Browser can use the compatible expanded video experience instead.
Cards, Security & Performance
FX
Smoother Site Cards Improved
Opening and closing animations have been redesigned and optimized. Site cards now feel smoother and more responsive, with less visual delay and better overall performance during launch, minimize, restore, and close transitions.
CF
Color Fill Animation New
Color Fill is a new selectable site-card loading animation. A droplet using the card's theme color falls into view and triggers a vibrant rising sequence of colors before the website is revealed. Choose it from Settings under Loading Animation for cards opened afterward.
S
Automatic Security-Challenge Return Improved
When a site requires a security challenge, Discovery Browser promotes it to a full browser window at the established fast timing. After verification succeeds, that temporary window closes and the original site is relaunched automatically in its normal card-style web view.
Media, Privacy & Help
M
New Media Player New
The rebuilt media player opens local audio and video files or entire folders, presents recently played media in a thumbnail grid, displays native video thumbnails and available audio album art, and provides playback, seeking, volume, playlists, visualizer, fullscreen, and compact floating playback controls.
AD
Discovery Ad Blocker New
A built-in ad blocker is now available at the top of the Add-ons area. Turn it on to block common advertising hosts, trackers, pop-ups, and advertising redirects without searching for or downloading a third-party extension.
?
Complete Help Center New
The new Help menu explains Discovery Browser features, everyday workflows, media tools, privacy behavior, settings, keyboard shortcuts, troubleshooting, Google sign-in limitations, updates, and support information. Topics are searchable so answers are easier to find.
Quality Improvements
OK
Bug Fixes & Reliability Fixed
Version 3.0.0 includes fixes across media controls, paused-media home behavior, floating playback, folder and recent-media thumbnails, audio artwork, playlist layering, pagination visibility, cinema compatibility notices, card behavior, and general interface reliability.
